## `templeit init`

```
USAGE
  $ npx templeit init [APP_NAME] --template=<nodejs|vuejs|reactjs>

ARGUMENTS
  APP_NAME  Name of your new application

FLAGS
  -t, --template=<value>  (required) What stack your new app is based

```
